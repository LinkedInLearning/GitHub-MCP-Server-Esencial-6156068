
# GitHub MCP Registry Esencial

Este es el repositorio del curso de LinkedIn Learning `[GitHub MCP Registry Esencial ]`. El curso completo está disponible en [LinkedIn Learning][lil-course-url].

![Nombre completo del curso][lil-thumbnail-url] 

Consulta el archivo Readme en la rama main para obtener instrucciones e información actualizadas.

DESCRIPCIÓN DEL CURSO

Descubre cómo integrar herramientas externas en tu entorno de desarrollo con GitHub Copilot usando el protocolo MCP. Este curso te guía paso a paso por los fundamentos técnicos de MCP, su implementación en GitHub, el uso del MCP Registry y el MCP Server. Aprende a conectar agentes de IA y potenciar tus flujos de trabajo como desarrollador frontend, backend o fullstack, aprovechando al máximo el poder de GitHub Copilot y la inteligencia artificial.

## Instrucciones

Este repositorio tiene ramas (branches) para cada uno de los vídeos del curso. Puedes usar el menú emergente de la rama en GitHub para cambiar a una rama específica y echar un vistazo al curso en esa etapa, o puedes añadir `/tree/nombre_de_la_rama` a la URL para ir a la rama a la que quieres acceder.

## Ramas

Las ramas están estructuradas para corresponder a los vídeos del curso. La convención de nomenclatura es Capítulo#_Vídeo#. Por ejemplo, la rama denominada `02_03` corresponde al segundo capítulo y al tercer vídeo de ese capítulo. Algunas ramas tendrán un estado inicial y otro resuelto.  

## Instalación

1. Para utilizar estos archivos de ejercicios, debes tener descargado lo siguiente:
   - Chrome
   - Visual Studio Code con GitHub copilot 
   - Node JS
   - Python (opcional)

2. Clona este repositorio en tu máquina local usando la Terminal (macOS) o CMD (Windows), o una herramienta GUI como SourceTree.
